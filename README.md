# G3D Phishing Tool

## نظرة عامة

أداة G3D هي أداة فخ (Phishing) تعليمية تُستخدم لاختبار الاختراق الأخلاقي (Penetration Testing). تتيح الأداة إنشاء صفحات تسجيل دخول مزيفة لمنصات شهيرة لاختبار الوعي الأمني لدى المستخدمين.

> **⚠️ تحذير قانوني:**
> استخدام هذه الأداة ضد أشخاص دون موافقتهم صريحة يُعتبر جريمة إلكترونية في معظم الدول. يجب استخدامها فقط في سياق اختبارات الاختراق المصرح بها.

## كيفية الاستخدام

### الخطوة 1: إدخال توكن ngrok

عند تشغيل الأداة لأول مرة، سيُطلب منك إدخال توكن ngrok الخاص بك.

- سجل حساباً مجاناً على [ngrok.com](https://ngrok.com)
- احصل على التوكن من لوحة التحكم
- ألصق التوكن في الطرفية

> يمكنك أيضاً تعديل المتغير `NGROK_TOKEN` في الكود مباشرة لتجنب الإدخال المتكرر.

### الخطوة 2: اختيار القالب

بعد الاتصال بنجاح، ستظهر قائمة بالقوالب المتاحة:

| الرقم | المنصة |
|-------|--------|
| 1 | فيسبوك (Facebook) |
| 2 | انستقرام (Instagram) |
| 3 | تويتر (Twitter) |
| 4 | سناب شات (Snapchat) |
| 5 | تيك توك (TikTok) |
| 6 | جيميل (Gmail) |
| 7 | ياهو (Yahoo) |
| 8 | لينكد إن (LinkedIn) |
| 9 | GitHub |
| 10 | Netflix |

### الخطوة 3: إرسال الرابط

بعد اختيار القالب، سيُولد رابط عام (Public URL) عبر ngrok. انسخ هذا الرابط وأرسله للهدف.

### الخطوة 4: مراقبة البيانات

اختر خيار **"بدء مراقبة البيانات"** لعرض أي بيانات يتم إرسالها عبر الصفحة المزيفة في الوقت الفعلي.

## البيانات المُلتقطة

لكل عملية تسجيل دخول، يتم حفظ:

- **الوقت والتاريخ**
- **عنوان IP**
- **الصفحة المصدر**
- **البيانات المُدخلة** (اسم المستخدم، كلمة المرور، إلخ)
